# v0.2 Scrum Notes and Project Kickoff Modes

v0.2 expands Context Capsule from single-task handoff packets into meeting and project-start planning packets.

The goal is not to evaluate teammates or assign work automatically. The goal is to turn meeting text into clear notes, decisions, action items, issue drafts, and team-lead review prompts.

## Scrum Notes Mode

Input:

- scrum notes
- meeting transcript
- instructor feedback
- optional project context

Output:

- source summary
- decisions
- blockers
- direction changes
- next actions
- open questions
- GitHub Issue drafts
- team-lead notes
- safety notes

CLI:

```powershell
.\context_capsule_cli.bat scrum-notes --text "Coach: Reduce MVP scope. Team: Build release notes." --json
```

Save Markdown:

```powershell
.\context_capsule_cli.bat scrum-notes --text-file .\meeting.txt --project-context "3rd project" --save --json
```

## Project Kickoff Mode

Input:

- contest or project topic
- idea meeting notes
- deadline
- constraints
- self-reported team context

Output:

- one-line pitch
- MVP scope
- out-of-scope list
- workstreams
- risks
- open questions
- GitHub Issue drafts
- submission checklist
- team-lead notes

CLI:

```powershell
.\context_capsule_cli.bat kickoff --topic "Scrum-to-execution planning tool" --notes "Build Scrum Notes Mode first. Discord API later." --deadline "2 weeks" --json
```

Save Markdown:

```powershell
.\context_capsule_cli.bat kickoff --topic "3rd project" --notes-file .\ideas.txt --deadline "2 weeks" --save --json
```

## Safety Boundary

Allowed:

- summarize meeting notes
- extract decisions and blockers
- create action items
- create issue drafts
- recommend what the team lead should ask next

Not allowed:

- score teammates
- infer skill from speaking volume
- auto-assign owners
- analyze hidden meeting recordings
- submit or deploy automatically

Team members should self-report capacity, preferred work, and blockers. Final assignment stays with the human team lead.

## Product Position

Short version:

```text
Turn scrum notes and kickoff ideas into reviewable work packets.
```

Portfolio version:

```text
Context Capsule v0.2 helps teams move from meeting discussion to execution by extracting decisions, risks, next actions, and issue drafts while keeping people evaluation and final assignment under human control.
```
