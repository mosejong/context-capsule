# Context Capsule v0.1.0

Context Capsule v0.1.0 is the first local-app release candidate for portfolio and presentation use.

## Highlights

- Local Windows launcher: double-click `run_context_capsule.bat`.
- Streamlit dashboard for generating target-specific handoff packets.
- CLI generate command for dashboard-free packet generation.
- GitHub Issue adapter with dry-run by default and explicit `--apply`.
- Saved output packets under `outputs/YYYYMMDD_HHMMSS_slug/`.
- AI, teammate, junior developer, and future-self handoff sections.
- Risk analyzer with mention risk and change risk separation.
- Token budget estimate with `approx_local_v1` verification status.
- Fixed login-error demo scenario.
- Performance comparison report and SVG.
- Release ZIP build script.

## Install

```text
Download context-capsule-v0.1.0.zip
Extract
Double-click run_context_capsule.bat
Open http://localhost:8501
```

Requirements:

- Windows 10/11
- Python 3.13
- Internet access for first dependency install

## CLI Demo

```powershell
.\context_capsule_cli.bat generate --repo-path . --task "Create a login API fix handoff packet" --target all --save --json
.\context_capsule_cli.bat create-issue outputs\YYYYMMDD_HHMMSS_slug --repo mosejong/context-capsule --json
```

`create-issue` is a dry-run unless `--apply` is passed.

These numbers are local estimates, not provider billing records.

## Security Model

- MVP features do not require external AI APIs.
- Repository context stays local unless the user explicitly sends it elsewhere.
- GitHub writes require explicit `--apply`.
- GitHub tokens are read from environment variables only.
- `.venv`, `outputs`, cache folders, build artifacts, and credentials are excluded from the release ZIP.

## Known Limits

- v0.1.0 is a Python/Streamlit local app, not a single executable.
- First run installs dependencies into `.venv`.
- Token savings are estimated locally.
- Discord adapter and external Token-analyzer adapter are not included yet.
- Chroma/hybrid RAG and local LLM adapters are planned for later releases.
