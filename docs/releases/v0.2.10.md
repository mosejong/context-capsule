# Context Capsule v0.2.10

v0.2.10 is an Evidence Persistence & Metric Conflict Guard release.

It responds to findings from a Raw vs Capsule README rewrite experiment:

- editable install was not ready because setuptools saw `outputs/` as a top-level package
- UI showed Token Evidence but saved `metadata.json` did not expose a matching `token_evidence` field
- Capsule preserved a source-backed `98.08%` metric, but the product did not warn when retrieved docs contained conflicting values such as `98.6%` and `98.08%`

## Download

```text
context-capsule-v0.2.10.zip
```

## What Changed

- `pip install -e .` now works by explicitly configuring setuptools package discovery.
- The installed CLI entrypoint works without `PYTHONPATH=.`:

```powershell
context-capsule doctor --repo-path . --json
context-capsule generate --repo-path . --task "README 포트폴리오 수치 확인" --save --json
```

- Saved `metadata.json` now includes a `token_evidence` object:
  - candidate file context tokens
  - retrieved context tokens
  - handoff prompt tokens
  - estimated reduction percent
  - method
  - verification status
  - actual provider usage status
- Risk analysis now flags documentation metric conflicts as `MEDIUM` mention risk.
- Added an Agent README Comparison spot-check report and SVG.
- Added a dummy FastAPI ecommerce repo 10-task spot-check report.

## Why It Matters

This patch moves Context Capsule closer to being usable by other people:

```text
No PYTHONPATH workaround
Token Evidence remains visible after saving
Conflicting README/QA metrics get surfaced before a user publishes a portfolio README
```

The Raw vs Capsule spot check is useful, but it is not a broad benchmark claim. It showed one important pattern:

```text
Raw README rewrite runs drifted to 98.6%.
Capsule-based runs preserved the source-backed 98.08%.
One Capsule run still invented file paths, so human review is still required.
```

The dummy-repo spot check also showed a useful product boundary:

```text
10 practical task requests.
Core target file included in 8/10 tasks.
Short requests such as pagination or unit-test creation still benefit from API path/file-name hints.
```
