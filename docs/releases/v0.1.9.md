# Context Capsule v0.1.9

## Release Asset

```text
context-capsule-v0.1.9.zip
```

## Release Type

Token Evidence UX patch.

v0.1.9 is based on KDT tester feedback after v0.1.8: the tool looked useful, but testers wanted to understand whether the generated prompt really reduces token usage.

## What's Improved

- Renamed the dashboard section from `Token Budget` to `Token Evidence`.
- Added an explanation of what is being compared:
  - candidate-file baseline
  - retrieved chunks
  - generated handoff prompt
- Added estimated saved-token text in the dashboard.
- Added a progress bar showing how large the handoff prompt is compared with the candidate-file baseline.
- Kept the honesty labels visible:
  - `Method: approx_local_v1`
  - `Verification status: Estimated only`
  - `Actual provider usage: Not measured yet`
- Added [Token Evidence Guide](../reference/token_evidence.md).
- Added Token Evidence guidance to `START_HERE_KO.md`.
- Added Token Evidence feedback field to the KDT beta quickstart.
- Regenerated the performance comparison report and SVG.

## Known Limits

- Token Evidence is still a local estimate, not provider billing usage.
- The future Token Analyzer adapter is still pending.
- Very small candidate contexts can show little or no estimated token reduction.
