# Context Capsule v0.2.9

v0.2.9 is a First Tester Orientation release.

It responds to tester feedback that the product worked, but first-time users still needed clearer guidance about where to start, what to inspect, and how the token-saving claim should be understood.

## Download

```text
context-capsule-v0.2.9.zip
```

## What Changed

- The first screen now says that most first tests should start from `AI에게 작업 맡기기`.
- Added a `무엇을 하고 싶나요?` guide below the mode cards.
- Added junior developer and interviewer/team lead viewpoints to the Work Handoff result summary.
- Added a dashboard explanation of how token context is reduced:
  - select relevant candidate files
  - compress them into a work handoff prompt
  - keep the number as a local estimate, not provider billing
- Updated Korean onboarding so testers do not try to read every result tab first.

## Why It Matters

Context Capsule should not only generate a correct result. It should also help a first-time user understand:

```text
어디서 시작하지?
나는 어떤 탭부터 봐야 하지?
이게 왜 토큰을 줄인다는 거지?
면접관/팀장은 이 결과에서 뭘 보면 되지?
```

v0.2.9 keeps the v0.2.8 guided result structure, but makes the path into that result easier.
