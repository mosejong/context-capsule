Context Capsule v0.2.1

대충 말한 작업을 실행 가능한 작업 패킷으로 바꿉니다.

외부 LLM 없이 로컬에서 레포와 회의록을 분석합니다. 자동 수정이나 자동 배정이 아니라, 사람이 승인할 수 있게 관련 파일, 위험, 다음 질문을 정리합니다.

처음 쓰는 순서
  1. 내 프로젝트 폴더 경로를 넣습니다.
  2. AI한테 말하듯이 요청을 적습니다.
  3. 생성 버튼을 누르고 결과 영역을 봅니다.

작업 하나 넘기기

“리드미 손보자”, “로그인 안돼”, “로컬 실행 안돼”처럼 입력하세요. 결과는 사람이 다 읽는 문서가 아니라 AI나 팀원에게 넘길 범위를 좁히는 자료입니다.

회의록 정리

스크럼이나 피드백 로그를 붙여넣으면 결정사항, 막힌 점, 다음 작업, 질문을 정리합니다.

프로젝트 시작 정리

아이디어 회의를 MVP 범위, 제외 범위, 작업 흐름, 제출 체크리스트로 정리합니다.

준비도 점검

회의록/스크럼/현재 상태를 넣으면 MVP와 프로토타입 준비도를 추정하고, 부족한 회의 항목과 다음 회의 질문을 제안합니다.

생성 중입니다.

요청을 분석하고 있습니다.

결과가 여기에 표시됩니다.

버튼을 누르면 이 영역에 요약, 관련 파일, 위험, 다음 질문이 나타납니다.
